Chapter 3: Data Representation & Boolean Logic

This chapter introduces essential concepts related to data representation in computers. It explains how computers use binary numbers (0s and 1s) to process and store information.

Key topics include number systems such as decimal, binary, octal, and hexadecimal, and how to convert between them. The chapter also covers how different types of data, like numbers, text, and images, are represented in memory. Additionally, Boolean logic, which plays a critical role in decision-making within computers, is introduced.

Review Questions & Answers

1. What is binary representation?

Binary representation is a number system using only 0s and 1s to store and process data.

2. Explain the importance of number systems in computing.

Computers rely on number systems (binary, octal, decimal, hexadecimal) to process and store data efficiently.

3. What are the different types of number systems?

The main types are decimal (base 10), binary (base 2), octal (base 8), and hexadecimal (base 16).

4. How do you convert decimal to binary?

To convert decimal to binary, repeatedly divide by 2 and record the remainders in reverse order.

5. What is Boolean logic?

Boolean logic is a form of algebra that uses true/false values and logical operators (AND, OR, NOT) for decision-making.

6. What are logic gates?

Logic gates are digital circuits that perform Boolean functions like AND, OR, NOT, XOR.

7. What is ASCII and why is it important?

ASCII (American Standard Code for Information Interchange) is a character encoding standard for representing text in computers.

8. What is Unicode?

Unicode is a universal character encoding system that supports multiple languages and symbols.

9. Explain the difference between RAM and ROM.

RAM is volatile memory used for temporary storage, while ROM is non-volatile memory containing permanent instructions.

10. What is a truth table?

A truth table is a table that shows the output of a logic function for all possible input combinations.

11. What is hexadecimal notation used for?

Hexadecimal is used in memory addressing and color representation in computing.

12. What is an example of a real-world Boolean logic application?

Boolean logic is used in search engines, circuit design, and decision-making systems.

13. How does a NOT gate work?

A NOT gate inverts the input, outputting the opposite value (1 becomes 0, and vice versa).

14. How is data stored in a computer?

Data is stored using binary representation in storage devices such as HDDs, SSDs, and RAM.

15. What is the difference between analog and digital data?

Analog data is continuous, while digital data is discrete and represented using binary values.

16. What is floating-point representation?

Floating-point representation is used to store real numbers in scientific notation for precision.

17. What are combinational circuits?

Combinational circuits perform logical operations where output depends only on current inputs.

18. What are sequential circuits?

Sequential circuits use memory to store past input states, affecting future outputs.

19. What is a half adder and a full adder?

A half adder adds two bits, while a full adder adds three bits (including carry).

20. Why do computers use binary instead of decimal?

Computers use binary because it simplifies circuit design and reduces errors in digital systems.
