Chapter 2: Computer System Organization

This chapter introduces the fundamentals of Computer System Organization, covering the basic structure of a computer, including the Central Processing Unit (CPU), memory, input/output devices, and storage systems. It emphasizes the role of hardware and software in executing tasks and the interaction between different components of the computer system.

The chapter also explains binary representation, which is crucial for understanding how data is processed by computers. Key concepts like data representation, logic gates, and the function of the arithmetic and logic unit (ALU) are introduced to provide a solid foundation for more advanced topics in computer science.

Review Questions & Answers

1. What are the main components of a computer system?

The main components include the CPU, memory (RAM & ROM), input devices, output devices, and storage devices.

2. What is the function of the Central Processing Unit (CPU)?

The CPU processes instructions, performs calculations, and controls other hardware components.

3. What are input and output devices? Give examples.

Input devices (keyboard, mouse) send data to the computer, while output devices (monitor, printer) display results.

4. What is primary memory? How is it different from secondary memory?

Primary memory (RAM, ROM) is used for temporary storage, while secondary memory (HDD, SSD) is used for long-term storage.

5. Explain the role of the ALU in a computer.

The Arithmetic Logic Unit (ALU) performs arithmetic and logical operations in the CPU.

6. What is binary representation?

Binary representation is a number system that uses only 0s and 1s to store and process data.

7. What are logic gates?

Logic gates are basic electronic circuits that perform logical operations like AND, OR, and NOT.

8. Define system software and application software.

System software (OS, utilities) manages hardware, while application software (MS Word, browsers) helps users perform tasks.

9. What is cache memory?

Cache memory is a small, high-speed memory inside the CPU that stores frequently used data for faster processing.

10. Explain the function of the motherboard.

The motherboard is the main circuit board that connects all components of a computer.

11. What is the difference between volatile and non-volatile memory?

Volatile memory (RAM) loses data when power is off, whereas non-volatile memory (ROM, HDD) retains data.

12. What is firmware?

Firmware is pre-installed software stored in ROM that helps hardware function properly.

13. What are buses in a computer system?

Buses are communication pathways that transfer data between CPU, memory, and peripherals.

14. Explain the role of the control unit in the CPU.

The control unit directs data flow, fetches instructions, and manages execution.

15. What is the difference between HDD and SSD?

HDDs use spinning disks for storage, while SSDs use flash memory, making them faster and more reliable.

16. What is an operating system?

An operating system manages computer hardware and software resources (e.g., Windows, Linux, macOS).

17. What is the function of the BIOS?

The BIOS (Basic Input/Output System) initializes hardware and loads the operating system.

18. What is the role of virtual memory?

Virtual memory extends RAM by using disk space to store overflow data temporarily.

19. What is a computer network?

A computer network is a system of connected devices that share resources and information.

20. What are the different types of storage devices?

Storage devices include HDDs, SSDs, USB flash drives, SD cards, and cloud storage.
