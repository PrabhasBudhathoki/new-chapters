Chapter 6: Web Technology I

This chapter introduces students to the fundamentals of web technologies used to create websites and web applications. It covers key components essential for web development, including HTML, CSS, JavaScript, and web hosting.

Students will also learn about the structure of web pages, the role of web browsers, and basic web security principles.

Review Questions & Answers

1. What is web technology?

Web technology refers to the tools and techniques used to develop and maintain websites and web applications.

2. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure web pages.

3. What is the role of CSS in web development?

CSS (Cascading Style Sheets) is used to style and format HTML elements, including layout, colors, and fonts.

4. What is JavaScript?

JavaScript is a programming language used to add interactivity to web pages, such as animations and dynamic content.

5. What are the differences between static and dynamic websites?

Static websites display fixed content, while dynamic websites generate content based on user interactions or databases.

6. What is a web browser?

A web browser is software that retrieves and displays web pages from the internet (e.g., Chrome, Firefox).

7. What is a web server?

A web server is a system that stores, processes, and delivers web pages to users.

8. What is responsive web design?

Responsive web design ensures that web pages adapt to different screen sizes and devices.

9. What is the function of HTTP and HTTPS?

HTTP (HyperText Transfer Protocol) is used for communication between web browsers and servers. HTTPS is a secure version of HTTP.

10. What is a URL?

A URL (Uniform Resource Locator) is the address used to access a webpage.

11. What is a domain name?

A domain name is a human-readable address used to access websites (e.g., www.example.com).

12. What is SEO?

SEO (Search Engine Optimization) is the process of improving a website's ranking on search engines.

13. What is web hosting?

Web hosting is a service that stores website files and makes them accessible online.

14. What are meta tags in HTML?

Meta tags provide metadata about a webpage, such as descriptions and keywords, for search engines.

15. What is the purpose of an SSL certificate?

An SSL certificate encrypts data to secure communication between users and websites.

16. What is a CMS?

A CMS (Content Management System) allows users to manage website content without coding (e.g., WordPress).

17. What are cookies in web technology?

Cookies are small data files stored on a user's computer to track website activity.

18. What is an API in web development?

An API (Application Programming Interface) allows web applications to interact with other services.

19. What is the difference between frontend and backend development?

Frontend development focuses on user interfaces, while backend development manages databases and server-side logic.

20. What are the advantages of using frameworks like Bootstrap?

Frameworks like Bootstrap simplify web design by providing pre-built styles and components.
